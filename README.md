## Functionalities 
### Users
:one: Used google api for join<br/>
:two: Can post pakages based one price, dates etc... <br/>
:three: Can delete posting recipies...<br/> 
:four: can see all posted recipies..<br/>
### Admins
:one: Used google api for join<br/>
:two: Can add pakages based one price, dates.<br/>
:three: Can delete recipies.<br/> 
:four: can see all posted recipies by all users<br/>
:five: can change status of all posted recipies which are requested from all users<br/>
